# TRAE FLOW 0.1.10

## 亮点

- 新增吉祥物动画速率调节：设置中可通过滑块调整宠物动画播放速度，支持 0 到 2 倍速，默认正常速度

## 修复

- 修复 Flow 岛收起后 WKWebView JS 挂起导致 Mineradio 自动切歌失效的问题：新增离屏窗口保活机制，收起时将 WebView 移入无焦点离屏窗口避免 macOS 挂起 JS
- 重构 Mineradio 桥接逻辑：使用 playQueue 获取可靠歌曲元数据替代 pendingSongId
- 优化歌词请求并发控制，修复切歌时歌词请求被阻塞的问题
- 移除不可靠的 MediaRemote 元数据回退，全部使用 WebView DOM 数据
- 增加元数据变更防抖，避免重复发送相同的歌曲信息

## 变更

- 更新 README 文案：定位调整为「macOS TRAE 任务灵动岛」，补充 NewsNow、Mineradio、桌面宠物等功能描述
- 升级 Mineradio 桥接协议版本到 1.4.0

## 说明

- 本次发版新增吉祥物动画速率设置，并重点修复 Flow 岛收起场景下 Mineradio 播放挂起的问题
