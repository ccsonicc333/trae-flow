# TRAE FLOW 0.2.2

## 亮点

- 新增左侧功能「位置式快捷展开」全局快捷键：默认 Option + 1~9 直接展开对应序号的已启用功能，修饰键模板可在设置中自定义
- 新增左侧功能独立快捷键：每个功能可绑定独立全局快捷键，设置页功能行内提供快捷键录制控件，nil 时回退位置式默认

## 变更

- `LeftFeature` 新增 `customShortcut` 字段，支持向后兼容解码
- `LeftFeatureStore` 新增 `setCustomShortcut` 接口
- `GlobalShortcutManager` 重构热键注册：支持 action / leftFeature / leftFeaturePositional 三类语义来源，Carbon 回调反查触发目标
- 设置页新增「左侧功能快速展开」快捷键项与功能行 `FeatureRowShortcutRecorder`
- `NotchView` 新增 `handleExpandLeftFeatureShortcut` 处理展开分发
- AGENTS.md 更新左侧功能与全局快捷键变更追踪说明

## 说明

- 本次发版主要新增左侧功能的位置式与独立快捷键体系，提升多功能场景下的切换效率
