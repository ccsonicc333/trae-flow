# TRAE FLOW 0.1.6

## 亮点

- 新增 Mineradio 电台功能及配套桥接实现，支持三平台登录状态检测与播放
- 左侧功能新增歌词显示与播放状态同步，紧凑视图支持歌词高亮与专辑封面展示
- 网页功能（webURL）新增自动获取网站 favicon，设置表单 URL 变更后自动拉取网站元数据
- 自动展开会话面板设置改为默认启用

## 修复

- 修复 Mineradio 切歌预缓冲导致的状态污染，重构消息处理逻辑
- 修复播放进度和歌词同步逻辑：不再使用 MediaRemote 的 elapsed/duration 数据，改为仅从 WKWebView 获取准确进度
- 修复 WKWebView 文件选择器不响应的问题
- 修复 LeftFeatureStore 拖拽排序的索引错位问题，改为基于 ID 的排序

## 说明

- 本次发版主要新增 Mineradio 电台功能并完善左侧功能的图标与歌词体验
- 设置菜单「左侧内容」更名为「左侧功能」
